# Olá, eu sou o Cauã Oliveira! 👋

Minha paixão pela tecnologia começou cedo, aos 11 anos, explorando a lógica por trás de sistemas e jogos. O que começou como curiosidade se transformou em uma carreira dedicada a construir arquiteturas sólidas, eficientes e de alta performance.

Como desenvolvedor, foco em unir a precisão da lógica de baixo nível (Pawn/C++) à agilidade do ecossistema Web moderno. Meu compromisso é resolver problemas complexos através de código limpo e soluções escaláveis.
